# Meridian-Landing-Page
Meridian is a clean, modern business landing page designed for a strategy and operations consultancy. It features a dark navy hero section with a bold headline, supporting stats, and clear calls to action, followed by a six-card services grid, a three-column testimonials section, and a structured contact form with a split layout.
The page uses a professional blue and navy color palette, smooth scroll-triggered fade-in animations, and a sticky navigation bar. It's fully responsive — the layout gracefully reflows to a single column on mobile, with fluid typography and touch-friendly tap targets throughout.
Overall, the design balances credibility and approachability: minimal enough to feel premium, structured enough to communicate trust.
